# Postshits da UEFS
- Este projeto foi um desafio pessoal de criar uma rede social utilizando somente uma única pagina para todo o sistema (desde o login ao perfil pessoal do usuário).
- Tem como tema principal posts em formato de postites, onde cada usuario pode criar posts com vários temas (avisos, anúncios, resenhas, etc...).
- Os usuários podem interagir votando positivo ou negativo para cada post e dependendo do seu rate, ele sobe ou desce.
## Infelizmente este projeto está parado devido a tecnologia utilizada, descidi mudar do PHP para React JS ;)
